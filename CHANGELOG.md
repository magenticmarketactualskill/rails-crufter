## [0.1.0] - 2024-12-06

### Added
- Initial release of rails-crufter gem
- Python bridge for atomic-cookie-crufter integration
- Template manager for applying templates
- File processor for extended naming conventions
- Generator base class for all generators
- Model generator with migration support
- Controller generator with views and helpers
- Scaffold generator for complete CRUD
- Extended file naming support (e.g., `File.html._styling._layout._content`)
- Automatic template tracking with atomic-cookie-crufter
- RSpec test suite
- Cucumber integration tests
- Comprehensive documentation

### Features
- Drop-in replacement for Rails generators
- Seamless Python 3.12 integration
- Template composition through extended naming
- Automatic mirror-based template tracking
- Support for all standard Rails generators
